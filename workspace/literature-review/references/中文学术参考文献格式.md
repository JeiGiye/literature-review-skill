# 中文学术参考文献格式

## 期刊论文

### 格式
作者. (年份). 题目. 期刊名, 卷(期), 起止页码. DOI

### 示例

**中文期刊：**
```
张三, 李四. (2020). 人工智能在医疗诊断中的应用研究. 中国生物医学工程学报, 39(2), 156-165.
```

**英文期刊：**
```
Smith, J., & Johnson, M. (2021). Deep learning for medical image analysis. Nature Medicine, 27(3), 456-468. https://doi.org/10.1038/s41591-021-01234-5
```

**三位及以上作者：**
```
王五, 赵六, 孙七, 周八. (2019). 基于卷积神经网络的肺结节检测算法. 计算机学报, 42(5), 876-885.
```

**团队作者：**
```
国家人工智能战略研究院. (2022). 人工智能伦理治理框架研究. 中国软科学, (3), 1-10.
```

## 学位论文

### 格式
作者. (年份). 题目[学位论文]. 授予单位.

### 示例

**博士学位论文：**
```
李华. (2020). 基于深度学习的医学图像分割方法研究[博士学位论文]. 清华大学.
```

**硕士学位论文：**
```
陈明. (2021). 自然语言处理技术在临床文本分析中的应用[硕士学位论文]. 北京大学.
```

---

## 会议论文

### 格式
作者. (年份). 题目. 见: 会议名称论文集. 起止页码.

### 示例

**中文会议：**
```
刘洋, 张伟. (2020). 智能医疗诊断系统的设计与实现. 见: 第十九届计算语言学大会论文集. 123-130.
```

**英文会议：**
```
Chen, L., & Wang, H. (2021). A survey on deep learning for medical image segmentation. In: Proceedings of the IEEE/CVF Conference on Computer Vision and Pattern Recognition. 2345-2354.
```

## 图书

### 格式
作者. (年份). 书名(版次). 出版地: 出版社.

### 示例

**中文图书：**
```
周志华. (2016). 机器学习. 北京: 清华大学出版社.
```

**英文图书：**
```
Goodfellow, I., Bengio, Y., & Courville, A. (2016). Deep learning. MIT press.
```

**译著：**
```
[美] 塞缪尔·亨廷顿. (1998). 文明的冲突与世界秩序的重建(周琪等译). 北京: 新华出版社.
```

---

## 图书章节

### 格式
章节作者. (年份). 章节题目. 见: 图书编者(编), 书名(版次, 起止页码). 出版地: 出版社.

### 示例
```
张明. (2019). 医学影像处理的深度学习方法. 见: 李华(编), 人工智能在医学中的应用(第2版, 45-68). 北京: 科学出版社.
```

## 网络资源

### 格式
作者. (发布日期). 题目[文献类型]. 网站名称. 检索日期, URL.

### 示例

**网页：**
```
世界卫生组织. (2023). 人工智能在医疗中的应用[EB/OL]. 检索于2023年5月10日, https://www.who.int/news/item/ai-in-healthcare
```

**技术报告：**
```
OpenAI. (2023). GPT-4 Technical Report[EB/OL]. 检索于2023年4月20日, https://arxiv.org/abs/2303.08774
```

## 专利

### 格式
专利所有者. (公开年份). 专利名称[专利类型]. 专利号.

### 示例
```
华为技术有限公司. (2021). 一种基于深度学习的医疗影像诊断方法[发明专利]. CN112234567A.
```

## 文献类型标识

- [M] - 专著
- [J] - 期刊文章
- [N] - 报纸文章
- [C] - 会议论文
- [D] - 学位论文
- [R] - 报告
- [S] - 标准
- [P] - 专利
- [EB/OL] - 电子公告/网络资源
- [DB/OL] - 数据库/网络资源

## 文内引用格式

### 单作者
```
张三(2020)指出...
```

### 双作者
```
张三和李四(2020)认为...
```

### 三位及以上作者
```
张三等(2020)的研究表明...
```

### 多篇文献同时引用
```
多项研究支持这一观点(张三等, 2020; Smith et al., 2021; 王五, 2022)
```

### 同作者同年多篇文献
```
李四(2020a, 2020b)分别从两个角度进行了探讨...
```

### 间接引用（二次引用）
```
Jones(2018)的研究(转引自Smith, 2020)表明...
```

### 直接引用（原文）
```
正如张三(2020)所指出的："人工智能正在改变医疗诊断的面貌"(p. 45)
```

## 注意事项

1. **作者姓名格式**
   - 中文作者：姓在前名在后，如"张三"
   - 英文作者：姓在前名在后，名用首字母，如"Smith, J."
   - 多作者之间用逗号分隔，最后两个作者之间用"&"（英文）或"、"（中文）

2. **年份格式**
   - 统一使用公历年份，如"(2020)"
   - 如文献无明确年份，使用"n.d."（no date）

3. **题目格式**
   - 中文题目不加引号，英文题目只有首字母和专有名词大写
   - 书名和期刊名使用斜体

4. **页码格式**
   - 期刊文章、图书章节必须标注起止页码
   - 使用"-"表示连续页码，如"123-130"

5. **DOI**
   - 英文期刊文章建议包含DOI链接
   - 使用https://doi.org/前缀

6. **排序规则**
   - 参考文献列表按第一作者姓氏拼音（中文）或字母（英文）排序
   - 同作者按年份排序（从早到晚）
   - 同作者同年按题目标题排序

7. **一致性**
   - 全文引用格式必须保持一致
   - 参考文献列表中的文献必须与正文中的引用一一对应
