# 文献管理工具指南

## 工具对比

| 工具 | 优势 | 适用场景 | 免费版 | 平台 |
|------|------|----------|--------|------|
| **Zotero** | 开源免费、插件丰富、易用 | 个人研究、学术写作 | 完全免费 | Win/Mac/Linux/Web |
| **EndNote** | 文献库强大、期刊格式丰富 | 大型研究团队、投稿多 | 功能受限 | Win/Mac |

## Zotero使用指南

### 安装与设置

1. **下载安装**
   - 访问 https://www.zotero.org/download/
   - 下载并安装Zotero客户端
   - 安装浏览器插件（Chrome、Firefox、Edge）

2. **账号注册**
   - 注册免费Zotero账号（300MB云存储）
   - 同步设置：Edit > Preferences > Sync
   - 输入账号信息，开启同步

### 文献导入

#### 浏览器插件导入
- 访问学术网站（CNKI、Web of Science等）
- 点击浏览器地址栏右侧的Zotero图标
- 选择保存位置（收藏夹）
- 自动抓取文献信息

#### 手动添加
- 文件 > 新建条目 > 选择文献类型
- 手动输入文献信息
- 拖拽PDF文件到Zotero自动识别

#### ISBN/DOI导入
- 工具 > 按标识符添加
- 输入ISBN（图书）或DOI（期刊文章）
- 自动检索并添加完整信息

#### CNKI批量导入
1. 在CNKI搜索文献
2. 勾选需要的文献
3. 点击"导出/参考文献"
4. 选择"EndNote"或"NoteExpress"格式
5. 在Zotero中选择"导入"（File > Import）

### 文献管理

#### 文献分类
- 创建收藏夹（New Collection）
- 建立层级结构（子收藏夹）
- 使用标签（Tags）标记文献
- 搜索功能（快速检索）

#### PDF管理
- 拖拽PDF到文献条目
- 自动关联或手动链接
- 内置PDF阅读器与标注
- 全文检索

#### 笔记功能
- 对每篇文献添加笔记
- 使用高亮和批注
- 笔记同步到云端
- 导出笔记到Word

### Word插件使用

#### 安装插件
- 在Zotero中选择"编辑 > 首选项 > 引用"
- 安装Word插件（自动检测Word）
- 重启Word即可使用

#### 插入引用
- 光标定位到插入位置
- 点击Word中的"Add/Edit Citation"图标
- 搜索文献并选择
- 选择插入位置（作者年份、数字等）

#### 生成参考文献
- 点击"Add/Edit Bibliography"图标
- 自动生成参考文献列表
- 参考文献列表自动更新

#### 切换引用格式
- 点击"Document Preferences"图标
- 选择引用格式（APA、GB/T 7714等）
- 如无所需格式，可在样式管理器中下载

#### 中文引用格式设置
1. 点击"Document Preferences"
2. 点击"样式管理器"（Style Manager）
3. 搜索"GB/T 7714"（中国国家标准）
4. 选择"Chinese Std GB/T 7714-2015 (numeric)"或"GB/T 7714-2015 (author-date)"
5. 点击"确定"应用

### 常用快捷键

- **Ctrl/Cmd + Shift + C**：添加新条目
- **Ctrl/Cmd + Shift + A**：添加笔记
- **Ctrl/Cmd + F**：搜索
- **Ctrl/Cmd + Shift + S**：同步

## EndNote使用指南

### 安装与设置

1. **安装EndNote**
   - 购买许可证或使用高校提供的授权
   - 下载并安装EndNote X9或EndNote 20
   - 创建个人Library（.enl文件）

2. **在线同步**
   - 注册EndNote Online账号
   - 设置同步：Edit > Preferences > Sync
   - 云端存储2GB，支持多设备同步

### 文献导入

#### 数据库导入
- Web of Science：直接导出到EndNote
- CNKI：选择"EndNote"格式导出，File > Import
- PubMed：选择"Send to Citation manager"

#### 手动添加
- References > New Reference
- 选择文献类型
- 填写文献信息

#### PDF导入
- File > Import > Folder
- 选择PDF文件夹
- EndNote自动识别PDF元数据

### Word插件使用

#### CWYW（Cite While You Write）
- 自动在Word中添加引用和参考文献
- 图标工具栏：插入引用、编辑引用、格式化参考文献

#### 插入引用
- 光标定位到插入位置
- 点击Go to EndNote图标
- 选择文献并点击Insert

#### 生成参考文献
- 点击Bibliography图标
- 选择输出样式（Output Style）
- 自动生成参考文献列表

#### 中文引用格式
1. 点击Bibliography下拉菜单
2. 选择"Select Another Style..."
3. 搜索"Chinese"或"GB"
4. 选择"Chinese Std GB/T 7714"

## Mendeley使用指南

### 安装与设置

1. **下载安装**
   - 访问 https://www.mendeley.com/download-reference-manager
   - 下载并安装Mendeley Desktop
   - 注册Mendeley账号

2. **设置同步**
   - 登录Mendeley账号
   - 设置同步文件夹（自动监控）

### 文献管理

#### 导入文献
- 拖拽PDF到Mendeley
- 使用Web Importer插件
- 从EndNote/Zotero导入

#### PDF标注
- 内置PDF阅读器
- 高亮、批注功能
- 导出标注到笔记

#### 文献发现
- Mendeley Catalog（文献数据库）
- 推荐相关文献
- 关注研究人员

### Word插件

#### 安装插件
- Tools > Install MS Word Plugin
- 重启Word

#### 使用方法
- Insert Citation：插入引用
- Insert Bibliography：生成参考文献
- Change Citation Style：切换引用格式

## 选择建议

### 个人用户
**推荐Zotero**
- 完全免费，无功能限制
- 插件生态丰富
- 易于上手
- 适合单机使用

### 高校学生/研究人员
**根据学校推荐**
- 国内高校多推荐NoteExpress
- 国际合作项目可考虑EndNote
- 重点关注中文支持

### 团队协作
**推荐Mendeley或EndNote**
- 强大的团队共享功能
- 在线协作
- 文献发现功能

### 轻量级使用
**推荐Zotero Web**
- 仅需浏览器插件
- 无需安装客户端
- 适合临时文献管理

## 常见问题

### 中文文献乱码
- **Zotero**：检查编码设置（Preferences > Advanced > Files and Folders）
- **EndNote**：安装中文字体，检查Import Filter
- **Mendeley**：使用最新版本

### 引用格式不对
- 检查是否选择了正确的输出样式
- 手动调整文献信息（作者、年份等）
- 考虑自定义样式

### PDF无法识别
- 确保PDF可复制文本（非扫描件）
- 使用OCR工具处理扫描件
- 手动补充元数据

### 同步问题
- 检查网络连接
- 确认账号密码正确
- 清除同步缓存

## 最佳实践

1. **定期备份**
   - 定期导出Library（.ris、.bib格式）
   - 备份PDF附件

2. **规范命名**
   - 文献标题清晰准确
   - 作者信息完整
   - PDF文件命名：作者年份_题名

3. **及时分类**
   - 导入文献后立即分类
   - 建立合理的收藏夹结构
   - 使用标签标记重要性

4. **定期整理**
   - 删除重复文献
   - 合并相似文献
   - 更新过时信息

5. **团队协作**
   - 共享Library（EndNote/Mendeley）
   - 统一引用格式
   - 明确分工与权限
