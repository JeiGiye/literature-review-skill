# 文献调研与综述

## 概述

本Skill提供完整的学术论文文献调研与综述撰写工作流，采用"多元文献检索+卡片化管理+述评引三步法"的方法论，严格遵循中文学术规范，确保综述具有批判性深度而非简单文献堆砌。

## 综述撰写规范

### 研究主题与规模要求

- **时间范围**：重点关注2024-2025年最新文献，兼顾少量开创性经典研究
- **总字数**：约3200字（±10%浮动范围）
- **引用文献**：34篇，文献分布如下：
  - 中文文献：12篇（占比35%，核心文献）
  - **英文文献**：22篇（占比65%，含Nature、PNAS、Computers & Education等顶级期刊）

### 文献结构要求

**中文文献（12篇核心文献）**
- 优先选择CSSCI、北大核心期刊
- 关注国家重点研发计划、国家自然科学基金资助项目
- 遴选高被引、高影响力、前沿研究成果

**英文文献（22篇）**
- **顶级期刊**（3-4篇）：Nature、Science、PNAS等综合类顶刊
- **权威期刊**（8-10篇）：Computers & Education（IF约4-5）、JASIST、MIS Quarterly、IEEE Transactions等
- **预印本与会议**（8-10篇）：arXiv、CHI、WWW、NeurIPS、ICML等
- 确保文献覆盖多个研究视角和方法论

### 文献结构要求

**中文文献（12篇核心文献）**
- 优先选择CSSCI、北大核心期刊
- 关注国家重点研发计划、国家自然科学基金资助项目
- 遴选高被引、高影响力、前沿研究成果

**英文文献（22篇）**
- 顶级期刊（Nature、Science、PNAS等）：3-4篇
- 领域权威期刊（Computers & Education、JASIST、MIS Quarterly等）：8-10篇
- 预印本与会议论文（arXiv、CHI、WWW等）：8-10篇
- 确保文献覆盖多个研究视角和方法论

## 工作流程

### 阶段1：多元文献检索与卡片化管理

#### 1.1 明确研究主题与关键词

- 确定研究问题的核心概念和边界
- 提取中英文关键词（包含近义词、上位词、下位词）
- 构建检索式并记录以便复现

#### 1.2 选择多元学术数据库

**中文数据库：**
- CNKI（中国知网）- 优先选择核心期刊、高被引文献
- 万方数据 - 补充文献来源
- 维普资讯 - 扩展检索范围
- 国家哲学社会科学文献中心 - 开放获取资源

**英文数据库：**
- **Semantic Scholar** - AI驱动的学术搜索引擎，提供引用图谱和影响因子
- **OpenAlex** - 开放学术图谱，覆盖全球学术文献和研究者网络
- Web of Science - 权威引文数据库，追踪研究脉络
- Scopus - 综合性文献数据库
- arXiv - 预印本平台，获取最新研究进展

**顶级期刊与会议：**
- Nature、Science、PNAS等综合类顶刊
- 领域权威期刊（如Computers & Education、JASIST、MIS Quarterly等）
- 顶级会议（如CHI、WWW、NeurIPS、ICML等）

#### 1.3 多元文献检索策略

**检索式构建：**
- 使用布尔运算符（AND、OR、NOT）组合关键词
- 限制发表年份：2024-2025年为主，兼顾经典研究
- 限制文献类型：期刊论文、会议论文、综述文章
- 设置语言过滤：中文、英文
- 使用Semantic Scholar的高级检索功能（领域、引用数、影响因子）

**检索优先级：**
1. 第一轮：Semantic Scholar + OpenAlex（获取广度）
2. 第二轮：CNKI + Web of Science（获取权威来源）
3. 第三轮：顶级期刊定向检索（确保高质量）

#### 1.4 文献筛选与卡片化管理

**筛选标准：**
- 快速浏览标题和摘要，排除无关文献
- 优先选择高被引、高影响因子期刊文献
- 确保覆盖不同研究方法、观点流派
- 中英文比例：中文12篇（35%），英文22篇（65%）

**卡片化管理（结构化元信息提取）：**
为每篇文献自动生成标准化的文献卡片，包含以下核心信息：

```
文献卡片模板
────────────────────────────────────
标题：[完整题名]
作者：[作者姓名]
年份：[发表年份，如2024]
来源：[期刊/会议名称，卷期页码]
DOI：[数字对象唯一标识符]
────────────────────────────────────
研究问题：[3-4句话概述研究目标]
研究方法：[定性/定量/混合研究方法]
研究样本：[样本规模、特征、来源]
主要发现：[关键研究结论，3-5个要点]
研究局限：[作者自述的局限性]
引用次数：[截至检索时的被引次数]
质量评级：[核心/权威/一般，判定依据]
────────────────────────────────────
标签：#主题 #方法 #流派 #争议
关联文献：[相关研究编号]
────────────────────────────────────
```

### 阶段2：文献深度阅读与核心发现提取

#### 2.1 精读核心文献

- 精读所有34篇文献的全文或高质量摘要
- 依据文献卡片信息，快速定位关键内容
- 重点关注研究方法、数据分析、结论推导过程

#### 2.2 分类整理文献

**按维度分类：**
- **研究主题/子领域**：将文献按研究问题归类
- **研究方法**：定性研究、定量研究、混合研究、实验研究、调查研究等
- **观点流派**：同一问题的不同理论观点和争议
- **时间演进**：2024-2025年的最新进展，与经典研究的对比

#### 2.3 核心发现提取（四大维度）

**维度1：效果上的争议**
- 识别不同研究对同一问题的结论差异
- 分析争议产生的原因（方法差异、样本差异、文化差异等）
- 总结主流观点和少数派观点
- 评估各方证据的强度

**维度2：技术上的演进**
- 梳理相关技术方法的发展脉络
- 识别技术突破点和瓶颈
- 分析新技术与传统方法的优劣对比
- 预测技术发展趋势

**维度3：形态的变革**
- 分析研究范式、理论框架的演进
- 识别跨学科融合趋势
- 总结实践形态的变化（如教育、管理、商业等领域）
- 探讨变革驱动力（技术、政策、社会等）

**维度4：研究空白与不足**
- 识别现有研究未覆盖的领域
- 指出研究方法的局限性
- 发现未被充分研究的群体或场景
- 提出未来研究方向

#### 2.4 建立文献关联图谱

- 基于引用关系构建文献网络
- 识别奠基性研究（高被引、高影响力）
- 发现研究流派和学术共同体
- 追踪观点的传播和演化

### 阶段3：述评引三步法撰写综述

#### 3.1 综述结构框架（总字数3200字）

**1. 引言（约320-480字，10-15%）**
- 研究背景与问题提出
- 研究意义与价值
- 文献检索范围与方法（说明34篇文献的来源和分布）
- 综述结构安排

**2. 研究现状梳理（约1600-1920字，50-60%）**
- 按研究主题/子领域组织段落
- 每段聚焦一个核心观点或争议
- 客观呈现各方观点与证据（每段引用3-5篇文献）
- 比较不同研究的异同，分析原因

**3. 核心发现分析（约640-800字，20-25%）**
- **效果争议**：总结不同研究结论的差异与原因
- **技术演进**：梳理技术方法的发展脉络和趋势
- **形态变革**：分析研究范式和实践形态的变化
- **研究空白**：指出未被充分研究的领域和问题

**4. 结论与建议（约160-320字，5-10%）**
- 总结全文要点和主要共识
- 提出未来研究建议和实践启示

#### 3.2 述评引三步法详解

**第一步：述（描述现状）**
- 客观描述研究现状和主要观点
- 准确呈现不同研究者的发现和结论
- 避免主观评价，基于事实陈述
- 引用关键文献支撑每项描述

**示例句式：**
- "张三（2024）的研究表明……"
- "多项研究（Smith et al., 2024; Johnson, 2025）证实了……"
- "李四等（2025）通过实证研究发现……"

**第二步：评（批判性分析）**
- 对述的内容进行分析和评价
- 比较不同研究的优劣、异同
- 指出研究的局限性和不足
- 识别争议背后的原因

**示例句式：**
- "然而，王五（2024）的研究发现与上述观点存在差异，这可能是因为……"
- "尽管Smith等（2025）的研究提供了有价值的证据，但其样本局限于……"
- "与国外研究不同，国内学者更关注……"

**第三步：引（引出方向）**
- 基于评述，引出研究空白
- 提出未来研究方向
- 给出实践启示和建议
- 避免简单的文献堆砌，形成有深度的综合评述

**示例句式：**
- "基于上述分析，未来研究可以从以下方向深入……"
- "现有研究尚未充分考虑……，这为后续研究提供了空间"
- "综上所述，企业信息安全建设应在……方面加强……"

#### 3.3 撰写技巧与注意事项

**段落结构：**
- 每段开头点明主题句（述）
- 中间展开论证，引用文献（述+评）
- 结尾总结提升，引出方向（引）

**逻辑衔接：**
- 段落之间使用过渡句
- 使用连接词（然而、此外、总之、值得注意的是）
- 保持逻辑链条清晰

**引用密度：**
- 每段引用3-5篇文献
- 避免过度引用（超过5篇）导致段落冗长
- 避免引用不足（少于3篇）导致论证单薄

**语言风格：**
- 使用第三人称，客观准确
- 学术化表达，避免口语化
- 专业术语首次出现时标注全称和缩写
- 符合中文学术规范

#### 中文学术规范要点

**引用规范（34篇文献）：**
- 使用"作者（年份）"格式进行文内引用
- 中文作者：首次引用列出全部作者（如"张三、李四、王五, 2024"），后续可用"张三等, 2024"
- 英文作者：3人以上用第一作者+et al.（如"Smith et al., 2024"）
- 同作者同年多篇文献：加字母区分（如"Smith, 2024a; Smith, 2024b"）
- 文内引用示例：多项研究支持这一观点(张三等, 2024; Smith et al., 2024; Johnson, 2025)

**参考文献格式（详见`references/中文学术参考文献格式.md`）：**

中文期刊论文：
```
张三, 李四. (2024). 企业信息安全管理研究进展. 中国管理科学, 32(4), 156-165.
```

英文期刊论文：
```
Smith, J., & Johnson, M. (2024). Data security governance in the AI era. Nature, 623(7986), 456-468. https://doi.org/10.1038/s41586-024-07654-x
```

预印本：
```
Johnson, R., Williams, L., & Brown, K. (2025). Zero-trust architecture for enterprise data protection. arXiv preprint arXiv:2502.12345.
```

会议论文：
```
Chen, L., & Wang, H. (2024). AI-driven data classification and protection. In: Proceedings of the 2024 CHI Conference on Human Factors in Computing Systems. 2345-2354.
```

**语言规范：**
- 使用第三人称，避免"我""我们"等第一人称
- 语言客观、准确、简洁
- 专业术语首次出现时标注全称和缩写（如"信息安全管理体系（Information Security Management System, ISMS）"）
- 避免口语化和过度修辞
- 符合中文学术写作语体

**格式规范：**
- 按自然段落写作，不使用分点列表
- 段落长度适中（200-300字为宜）
- 段落之间使用空行分隔
- 不使用Markdown加粗、斜体等格式（除非强调术语）

#### 写作技巧（基于述评引三步法）

**1. 逻辑清晰（述）**
- 每段开头点明主题句，清晰表达本段核心观点
- 段落之间使用过渡句，形成逻辑链条
- 使用连接词衔接：此外、然而、因此、值得注意的是、综上所述

**2. 批判性分析（评）**
- 不仅陈述观点，要分析其合理性与局限性
- 比较不同研究的异同，分析差异产生的原因
- 指出研究方法、样本、结论的可推广性
- 基于文献证据提出自己的见解（非主观臆断）

**3. 引出方向（引）**
- 基于评述内容，自然引出研究空白
- 提出具体的未来研究方向
- 给出实践启示和建议
- 避免简单的文献堆砌，形成有深度的综合评述

**4. 避免常见问题**
- ❌ 简单罗列文献，缺乏分析整合（只有述，没有评和引）
- ❌ 过度引用，每段引用超过5篇，导致段落冗长
- ❌ 偏向某一观点，忽视对立研究，缺乏平衡性
- ❌ 引用不规范，文献格式不统一，难以追溯来源
- ❌ 缺乏批判性思维，仅做文献的搬运工，没有自己的分析
- ❌ 研究空白识别不准确，或提出的建议过于泛泛

## 使用示例

**用户请求示例：**
> "帮我调研关于企业信息安全与数据安全治理的文献，写一篇综述（3200字，34篇文献）"

**Agent执行流程：**

### 1. 多元文献检索
- 确定研究主题：企业信息安全管理、数据安全治理
- 提取关键词：信息安全、数据安全、风险管理、治理框架、零信任架构、AI安全
- 多数据库检索：
  - Semantic Scholar（获取广度和影响因子）
  - OpenAlex（追踪研究脉络）
  - CNKI（中文核心期刊）
  - Web of Science（权威来源）
  - Nature、PNAS、Computers & Education等顶级期刊定向检索

### 2. 文献筛选与卡片化管理
- 检索范围：2024-2025年为主，兼顾经典研究
- 筛选结果：中文12篇（核心），英文22篇（含Nature 2篇、PNAS 1篇、Computers & Education 1篇）
- 为每篇文献生成标准化卡片，自动抽取元信息：
  ```
  标题：Zero-trust architecture for enterprise data protection
  作者：Johnson, R. et al.
  年份：2024
  来源：Nature, 623(7986), 456-468
  DOI：10.1038/s41586-024-07654-x
  引用次数：127（截至2025年2月）
  质量评级：顶级（Nature期刊）
  标签：#零信任 #架构 #数据保护
  ```

### 3. 深度阅读与核心发现提取
- 精读34篇文献，按维度分析：
